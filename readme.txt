== Installation ==
Step 1:
= A step-by-step guide to installing a MaxelPay crypto payment plugin on WooCommerce =

1. From your WordPress admin panel, go to Plugins> Add New>Search the MaxelPay plugin in Search

2. Install and activate the MaxelPay plugin.

3. Get the API key And API Secret from your MaxelPay merchant dashboard.

4. Save the key in your MaxelPay payment settings, you’re ready to go.

Step 2:

1. Go to the WordPress.org website and download the MaxelPay plugin.
2. After Downloading, go to the admin panel and upload the unzipped folder to the /wp-content/plugins on your website.
3. Next, you need to activate it. Click on Plugins, then find MaxelPay Plugin and click Activate.
4. After activating, you can set up MaxelPay by going to the WooCommerce Payments section in your dashboard.

== Frequently Asked Questions ==

= How does the MaxelPay plugin work? =

The MaxelPay plugin enables WordPress-based websites and e-commerce stores to accept cryptocurrency payments. When you install and activate it on your website, it integrates with WooCommerce, allowing customers to pay using a variety of cryptocurrencies.

= How can I set up the MaxelPay plugin on my WordPress website to start accepting cryptocurrency payments? =

To set up the MaxelPay plugin:
1. Install and activate it on your WordPress site.
2. Configure the settings by adding your Payment API key, Secret key,  and other relevant details for processing transactions.


= Can I accept cryptocurrency without registering for MaxelPay?: =
No, you cannot. To start accepting payments, you need to [register](https://dashboard.maxelpay.com/register?utm_source=wordpress&utm_medium=referral&utm_campaign=wordpress_plugin) for MaxelPay. 

= What credentials do I need to accept cryptocurrency payments on my WordPress website? =

You will need to enter your API key and Secret key to accept cryptocurrency payments on your WordPress site. To generate your MaxelPay API key and API Secret,  [click here](https://www.maxelpay.com/documentation/integration?utm_source=wordpress&utm_medium=referral&utm_campaign=wordpress_plugin).

= Which cryptocurrency wallet do you support? =
We support wallets like MetaMask, WallConnect, Phantom, CoinBase, and more.

= Do I need to install the WooCommerce plugin to use MaxelPay? =
Yes, you must install WooCommerce (version 9.0.2 or later) before using the MaxelPay plugin.


= Can you help me to integrate cryptocurrency payments into my website? =
Of course, our support team is always here to help. Contact us by email at  support@maxelpay.com.
